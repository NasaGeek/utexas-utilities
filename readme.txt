In order to compile this project you'll need to grab a few libraries:

ActionBarSherlock
PanesLibrary (with modifications)
SlidingMenu
Google Play Services
AndroidPlot (just a jar)
Crittercism (just a jar)
AmazingListView (with modifications)
ViewPagerIndicator (with modifications)
Android Support Library (just a jar)

UTilities should compile fine with the most recent versions of all of these libraries after the noted libraries have been modified
